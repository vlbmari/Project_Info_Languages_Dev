Base de Conhecimento

Entenda sobre tipos de execução:

Compilada

O código-fonte é traduzido para código de máquina antes da execução, resultando em programas mais rápidos.

Interpretada

O código é lido e executado linha por linha por um programa (interpretador) em tempo real.

Interpretada (JIT)

Uma abordagem híbrida que compila partes do código durante a execução para otimizar a performance.

Traduzida (Transpilada)

O código-fonte é convertido para outro código-fonte de nível similar, como TypeScript para JavaScript.

Compilada para bytecode / IL

O código é compilado para um formato intermediário, que é executado por uma Máquina Virtual (VM).
