Linguagens de programação

Entenda sobre tipos de execução:

Compilada

O código é traduzido de uma só vez para a linguagem da máquina antes da execução.

Interpretada

O código é lido e executado linha por linha em tempo real.

Interpretada (JIT)

Começa interpretando e compila as partes mais usadas em tempo real para otimizar a velocidade.

Traduzida (Transpilada)

O código é convertido para outra linguagem de programação (ex: TypeScript para JavaScript).

Compilada para bytecode

O código é compilado para um formato intermediário universal, específico da plataforma Java.

Compilada para IL

O código é compilado para a "Intermediate Language", formato universal do ecossistema .NET.

Entenda sobre Níveis de Linguagens:

Nível Alto

Mais próximas da linguagem humana, fáceis de aprender e usar, com forte abstração do hardware.

Nível Intermediário

Uma ponte entre o alto e o baixo nível, permitindo manipulação de memória e acesso ao hardware.

Nível Baixo

Muito próximas da linguagem da máquina, oferecendo controle máximo sobre o hardware, mas com alta complexidade.
